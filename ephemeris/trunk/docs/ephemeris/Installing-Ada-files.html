3.1.1 Downloading and compiling the AEL distribution

The AEL can be dowloaded from http://www.dit.upm.es/jpuente/AEL as a set of Ada source and other auxiliary files packed in a .tgz (for UNIX or GNU/Linux systems) or .zip (for Windows® systems) archive.

An Ada 2005 compilation system must be installed in order to compile the library. GNAT GPL 2006 is assumed in the following, but any other Ada compilation system can be used as well. In this case, refer to the appropriate manuals for compilation instructions.

To compile the library on a UNIX or GNU/Linux system, unzip the archive into any directory (<build-dir>) and then type: 

     $ cd <build-dir>
     $ make
