2.2 The JPL ephemeris files

The JPL ephemeris files contain the Chebyshev coefficients for the position and velocity of the main solar system bodies across a given time span. The coefficients are grouped into records, each of them containing data for a 32-day long interval. Intervals are further subdivided into granules, with a length that depends on the body under consideration. The coefficients enable the position and velocity components to be computed for each major solar system body. Ephemeris file formats, contains a detailed description of the JPL source files and the binary files used by AEL.
