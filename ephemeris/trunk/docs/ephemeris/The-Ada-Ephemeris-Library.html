2.3 The Ada Ephemeris Library

The Ada Ephemeris Library is a set of Ada packages that provide high-level access to the state of the main solar system bodies: Mercury, Venus, the Earth, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto, the Moon, and the Sun1. The position and velocity values are given in AU2 and AU/day3. Notice that the coefficients in the ephemeris files give the state components in km and km/day, respectively. The unit transformations are carried out internally in the AEL operations.

The main packages in the AEL are Ephemeris and Ephemeris.Data. The former is the root package of the library, in which the supported ephemeris sets (DE200, DE405, and DE406)4 are defined. The Ephemeris.Data package contains all the necessary definitions to get the state of a body at a given time. The next chapter (see User's Guide) contains a detailed description of these two packages.

Footnotes

[1] The ephemeris files contain data for a slightly different set of bodies: Mercury, Venus, the Earth/Moon barycenter, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto, the Moon (in geocentric coordinates), and the Sun, plus nutation angles and, in some cases, libration angles. The transformations for the Earth and the Moon are carried out internally in the AEL operations, and the nutation angles are not being used in this version of AEL.

[2] An AU (Astronomical Unit) is a unit of distance which equals 1.4959787066 \times 10^11 m.

[3] The astronomical day is defined as 86400 s.

[4] Notice that the current version has only been tested with DE200
