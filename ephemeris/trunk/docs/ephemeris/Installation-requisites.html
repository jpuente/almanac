1.2 Installation requisites

The AEL software is highly portable and can be used on any platform for which an Ada 2005 compilation system is available. The distributed version has been compiled with GNAT Pro 5.04 and GNAT GPL 2005 on PC workstations running GNU/Linux and Windows® XP operating systems. Compiling the software with other Ada 2005 compilers should no give any problems as only Ada 2005 standard language features have been used.
