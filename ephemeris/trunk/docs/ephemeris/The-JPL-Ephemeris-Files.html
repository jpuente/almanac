1.2.1 The JPL Ephemeris Files

The JPL ephemeris files contain high-precision data that have been computed by integrating the equations of motion for the main solar system objects with respect to the barycentric J2000.0 reference frame.

Sun, Moon, planets, nutation (libration).

Since providing the values of the state components (position and velocity) for every point in time is not possible, the ephemeris provide instead a set coefficients that can be used to calculate the state at a given point of time. To this point, time is divide into continous adjacent intervals called granules. The state components as functions of time are then expressed as a series development of Techbyshev polynomials:

pi(t) = ... vi(t) = ..

Where t is the normalized time (-1.0..+1.0) in a granule span. The JPL ephemeris files contain the position and velocity coefficients for every granule in the ephemeris time span and for everybody. From these coefficients and the Techbyshev polynomial coefficients the state components a t a given time can easily be computed.

JPL ephemeris files are distributed both in source (ASCII) and binary formats. Although both formats contain the same information, binary files are preferrable as they are more compact and have a shorter access time. Unfortunately, the binary files available at the JPL server can only be used with a restricted set of platforms and are not easiliy portble. For this reason, the Ada Ephemeris Library uses binary files which are generated from the JPL source files with a specific utility program called create-ephemeris (see chapter xxx).

In order to get a compact representaion, intyerpolation coefficients are stored in binary files in a specific format (see appendix xxx) which is not directly readable. The AEL packages provide a seof procedures that extract the set of coefficients for the granule that contains the time instant for which a body state is sought, and then compute the position and velocity values using the polynomial interpolation formulas xxx.

There are several sets of ephemeris files, identified by the acronyms DE200

DE200 : (includes nutations but not librations)

JED 2305424.5 (1599 DEC 09) to JED 2513360.5 (2169 MAR 31)

This ephemeris has been the basis of the Astronomical Almanac since 1984. It is based upon the dynamical equator and equinox of J2000 (see Standish, 1982 and Standish, 1990).

DE405 : (includes both nutations and librations)

JED 2305424.50 (1599 DEC 09) to JED 2525008.50 (2201 FEB 20)

Our latest ephemeris; it is based upon the International Celestial Reference Frame (ICRF). (DE200 is within 0.01 arcseconds of the frame of the ICRF). DE405 was created in May-June 1997.
