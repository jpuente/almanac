1.3 Acknowledgments

The information on the JPL ephemerides that has been used to develop the Ada Ephemeris Library comes from the following sources:

  • The JPL ephemerides package user's guide1.
  • David Hoffman's user's guide for the C utility programs for processing JPL Ephemeris Data2.
  • The Explanatory Supplement to the Astronomical Almanac, edited by P. Kenneth Seidelmann, University Science Books, 1992, especially chapter 5.
